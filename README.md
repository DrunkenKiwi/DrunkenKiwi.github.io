# Le bloc note du kiwi bourré

## Le blog / bloc note sans trop d’intérêt d'un développeur, cinquantenaire, père de famille, râleur, anarchiste sur les entournures, amateur de rhum devant l’éternel et les hommes.

Autant le dire tout de go, je ne suis pas un grand écrivain. Si j’avais des dons de plume, je ne me serais jamais lancé dans un blog, j’aurais écrit un joli roman que j’aurais publié chez un bel auteur parisien et je passerais le reste de mes jours à siroter du rosé en regardant la garrigue s’étaller au-dessus de mes orteils.
